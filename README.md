# AI Urbanistica
Applicazione RAG (Retrieval-Augmented Generation) per l'interrogazione delle normative urbanistiche italiane a livello Nazionale, Regionale, Provinciale e Comunale. 

Realizzata con FastAPI (Backend) e React+Vite (Frontend).

## Requisiti
Affinché il sistema di AI funzioni, assicurati di aver configurato il secret `OPENAI_API_KEY` nelle impostazioni di questo Space su Hugging Face.
